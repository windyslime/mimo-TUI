# 任务 9: 子智能体监控视图

> 先阅读 `00-项目总览.md` 了解项目背景，然后执行本任务。

## 要求

实现 SubAgents 视图，用于监控子智能体的状态。

### 1. 视图布局

```
┌── SubAgents (3 active) ──────────────────────────┐
│                                                    │
│  ID              Role          Status     Progress │
│  ─────────────   ──────────    ───────    ───────  │
│  abc-123         Explore       🔄 Running  ███░░ 60%│
│  def-456         Implementer   ✅ Done     █████ 100%│
│  ghi-789         Plan          ⏳ Pending  ░░░░░  0%│
│                                                    │
│  [r] Refresh  [c] Cancel  [d] Details  [q] Back   │
└───────────────────────────────────────────────────┘
```

### 2. 功能

- 实时刷新子智能体状态
- 查看单个子智能体的详细信息（任务描述、输出日志）
- 取消正在运行的子智能体
- 查看子智能体结果

### 3. 子智能体类型参考

```rust
pub enum SubAgentRole {
    General,      // 全能型
    Explore,      // 只读探索
    Plan,         // 任务规划
    Review,       // 代码审查
    Implementer,  // 代码实现
    Verifier,     // 测试验证
    Custom,       // 自定义
}

pub enum SubAgentStatus {
    Pending,
    Running,
    Completed,
    Failed,
    Interrupted,
    Cancelled,
}
```

## 交付物

- 创建 `packages/apps/tui/src/views/agents.rs`：子智能体监控视图
- 支持刷新、查看、取消操作
- `cargo check` 编译通过
