# 任务 13: 键盘导航与滚动

> 先阅读 `00-项目总览.md` 了解项目背景，然后执行本任务。

## 要求

### 1. 消息区滚动

| 按键 | 行为 |
|------|------|
| `j` / `↓` | 向下滚动一行 |
| `k` / `↑` | 向上滚动一行 |
| `g` | 跳到顶部 |
| `G` | 跳到底部 |
| `Ctrl+D` | 向下半页 |
| `Ctrl+U` | 向上半页 |

### 2. 列表导航（会话列表、工具列表等）

| 按键 | 行为 |
|------|------|
| `j` / `k` | 上下移动选中项 |
| `Enter` | 选择当前项 |
| `Esc` | 返回列表/返回上一层 |

### 3. 鼠标支持（可选）

- 滚轮滚动消息区
- 点击切换视图标签

### 实现方式

```rust
pub struct ScrollState {
    pub offset: usize,        // 当前滚动偏移
    pub max_offset: usize,    // 最大可滚动偏移
    pub viewport_size: usize, // 可视区域大小
    pub content_size: usize,  // 内容总大小
}

impl ScrollState {
    pub fn scroll_down(&mut self, lines: usize);
    pub fn scroll_up(&mut self, lines: usize);
    pub fn scroll_to_top(&mut self);
    pub fn scroll_to_bottom(&mut self);
    pub fn clamp(&mut self);  // 确保 offset 在合法范围内
}
```

### 与 App 集成

在 App 结构中添加滚动状态：

```rust
pub struct App {
    // ... 现有字段
    pub scroll_state: ScrollState,
}
```

在 Normal 模式下处理 `j`/`k` 等导航键。

## 交付物

- 在 App 中添加滚动状态管理
- 在各个视图中处理导航键
- `cargo check` 编译通过
