# 任务 12: 标题栏与状态栏

> 先阅读 `00-项目总览.md` 了解项目背景，然后执行本任务。

## 要求

### 1. 标题栏（Header）

```
MiMo-TUI                                    [MiMo v2.5] [🧠 ON]
```

- 左侧：应用名称
- 右侧：当前模型、记忆状态

### 2. 状态栏（Footer）

```
[Chat] [Agents(2)] [History]           Normal  |  q:Quit  ?:Help
```

- 左侧：视图切换标签（当前视图高亮）
- 中间：当前模式（Normal / Insert / Command）
- 右侧：快捷键提示

### 3. 动态状态消息（覆盖在 Footer 上方）

```
✓ Memory added: "prefer pytest"
✗ API Error: rate limit exceeded
⏳ Connecting to MiMo API...
```

- 3 秒后自动消失
- 支持 Info、Success、Warning、Error 类型
- 样式区分：
  - ✅ Success：绿色
  - ℹ️ Info：蓝色
  - ⚠️ Warning：黄色
  - ❌ Error：红色

### API 设计

```rust
pub struct StatusBar {
    message: Option<String>,
    message_type: StatusType,
    timestamp: Option<Instant>,
    duration: Duration,  // 默认 3 秒
}

impl StatusBar {
    pub fn show(&mut self, message: &str, status_type: StatusType);
    pub fn update(&mut self);  // 检查是否需要隐藏过期消息
    pub fn render(&self, f: &mut Frame, area: Rect);
}
```
