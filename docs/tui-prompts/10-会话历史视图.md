# 任务 10: 会话历史视图

> 先阅读 `00-项目总览.md` 了解项目背景，然后执行本任务。

## 要求

实现 History 视图，用于查看和切换会话。

### 1. 视图布局

```
┌── Sessions ───────────────────────────────────────┐
│                                                    │
│  Created         Title                  Messages   │
│  ─────────────   ──────────────────────   ───────  │
│  10:30           重构用户认证模块            12     │
│  09:45           分析数据库连接池            8     │
│  昨天            编写单元测试               24     │
│                                                    │
│  [Enter] Open  [d] Delete  [q] Back               │
└───────────────────────────────────────────────────┘
```

### 2. 功能

- 列出所有历史会话
- 使用第一消息作为会话标题预览（Session 已有 `preview()` 方法）
- 切换到之前的会话
- 删除不需要的会话

### 3. Session API 参考

```rust
// 从 mimo_core::SessionManager 获取：
fn list_sessions(&self) -> Vec<&Session>;
fn get_session(&self, id: &str) -> Option<&Session>;
fn delete_session(&mut self, id: &str);

// Session 方法：
fn preview(&self) -> String;  // 返回第一条消息的前 120 字符
```

## 交付物

- 创建 `packages/apps/tui/src/views/history.rs`：会话历史视图
- 支持列表、切换、删除操作
- `cargo check` 编译通过
