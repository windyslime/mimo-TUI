### 2. 各区域约束

| 区域 | 约束 | 说明 |
|------|------|------|
| Header | 固定 1 行 | 应用名 + 模型 + 记忆状态 |
| Messages | `Min(1)` | 占据剩余空间 |
| SubAgents | 固定宽度 20-25 列 | 子 Agent 状态面板 |
| EditTracker | 动态高度 5-8 行 | AI 编辑追踪面板 |
| Input | 固定 3 行 | 多行输入可扩展 |
| Footer | 固定 1 行 | 视图标签 + 模式 + 快捷键 |

### 3. 布局响应式

- 终端宽度 < 100 列：隐藏 SubAgents 面板（按 Tab 切换查看）
- 终端宽度 < 80 列：隐藏 EditTracker 面板
- 终端大小变化时自动重排

### 4. 焦点高亮

当前聚焦的区域用不同边框颜色/样式标识

### 5. 使用 `ratatui::layout` 的 `Layout` + `Constraint` 实现

## 新增板块一：SubAgents 子 Agent 活动面板

### 功能定位

实时展示各子 Agent 的运行状态、任务进度、已完成操作及关键执行结果。让用户随时了解后台并行任务的进展。

### 布局设计

```
┌── SubAgents (3) ─────────────────────┐
│                                       │
│ 🔄 explore-01    代码探索              │
│    Analyzing src/engine/...          │
│    ████░░░░ 60%                      │
│                                       │
│  impl-02      代码实现              │
│    Writing utils.rs                  │
│    ███░░░░░ 40%                      │
│                                       │
│ ✅ plan-03      任务规划              │
│    Done: 3 steps defined            │
│    ████████ 100%                     │
│                                       │
│ [r] Refresh  [d] Details  [q] Back   │
└───────────────────────────────────────┘
```

### 状态指示器

| 图标 | 状态 | 颜色 | 说明 |
|------|------|------|------|
|  | Pending | 灰色 | 等待调度 |
| 🔄 | Running | 蓝色 | 正在执行 |
|  | ToolCall | 黄色 | 调用工具中 |
| ✅ | Completed | 绿色 | 已完成 |
| ❌ | Failed | 红色 | 执行失败 |
|  | Cancelled | 灰色 | 已取消 |

### 进度条设计

```
░░░░░░░░  0%    未开始
██░░░░░░  25%   刚开始
████░░░░  50%   进行中
██████░░  75%   接近完成
████████  100%  已完成
```


## 整体布局实现方案

### Layout 结构

```rust
fn build_layout(area: Rect) -> Layout {
    Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(1),   // Header
            Constraint::Min(1),      // Main Content (Messages + Side Panels)
            Constraint::Length(3),   // Input
            Constraint::Length(1),   // Footer
        ])
        .split(area)
}

fn build_main_layout(area: Rect) -> Layout {
    // 根据终端宽度决定是否显示侧边栏
    if area.width >= 120 {
        Layout::default()
            .direction(Direction::Horizontal)
            .constraints([
                Constraint::Percentage(55),  // Messages Area
                Constraint::Percentage(25),  // Edit Tracker
                Constraint::Percentage(20),  // SubAgents Panel
            ])
            .split(area)
    } else if area.width >= 90 {
        Layout::default()
            .direction(Direction::Horizontal)
            .constraints([
                Constraint::Percentage(70),  // Messages Area
                Constraint::Percentage(30),  // Edit Tracker (SubAgents hidden)
            ])
            .split(area)
    } else {
        Layout::default()
            .direction(Direction::Vertical)
            .constraints([
                Constraint::Percentage(80),  // Messages Area
                Constraint::Percentage(20),  // Edit Tracker (compact)
            ])
            .split(area)
    }
}
```

### 焦点管理

```rust
pub enum FocusTarget {
    Messages,
    Input,
    SubAgents,
    EditTracker,
}

impl App {
    pub fn cycle_focus(&mut self) {
        self.focus = match self.focus {
            FocusTarget::Messages => FocusTarget::Input,
            FocusTarget::Input => FocusTarget::SubAgents,
            FocusTarget::SubAgents => FocusTarget::EditTracker,
            FocusTarget::EditTracker => FocusTarget::Messages,
        };
    }
}
```

### 渲染优先级

1. 始终渲染：Header、Messages、Input、Footer
2. 宽度 >= 90：额外渲染 EditTracker
3. 宽度 >= 120：额外渲染 SubAgents Panel
